skills

  • HTML5
    Язык для структурирования и представления содержимого всемирной паутины.
    Предназначенная для создания веб-приложений использующих аудио, видео, графику, анимацию и многое другое.
  • CSS3
    Формальный язык описания внешнего вида документа (веб-страницы).
    Используется для стилизации веб-страницы.
    Позволяет применять стили выборочно к элементам в документах HTML.
  • JavaScript
    Язык программирования.
    Поддерживает объектно-ориентированный, императивный и функциональный стили. Является реализацией спецификации ECMAScript.
    Используется для внедрения интерактивности на Web - страницу.
  • SASS / SCSS
    Метаязык на основе CSS, предназначенный для увеличения уровня абстракции CSS-кода и упрощения файлов каскадных таблиц стилей.
  • GIT
    Инструмент, позволяющий реализовать распределённую систему контроля версий.
    Даёт возможность разработчикам отслеживать изменения в файлах и работать над одним проектом совместно с коллегами.
  • Visual Studio Code
    Редактор исходного кода. Редактор кода для кроссплатформенной разработки веб- и облачных приложений. Имеет широкие возможности для кастомизации и множество инструментов.
  • npm
    Стандартный менеджер пакетов, автоматически устанавливающийся вместе с Node.js. Он используется для скачивания пакетов из облачного сервера npm, либо для загрузки пакетов на эти сервера.
  • gulp
    Таск-менеджер для автоматического выполнения часто используемых задач, написанный на языке программирования JavaScript.
